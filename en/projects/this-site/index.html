This Site

October 2020 - Present

Project's website

Context

Creation of this site to introduce myself and share my passions.

My first desire was to showcase my skills in a more controlled way than, for example, on LinkedIn. Similarly, in order to efficiently centralize the articles I have written on Medium or Talan Labs' blog, I needed a blog.

Technical Solution

The Tool

This site is designed using a static site generator, Hugo, and hosted for free thanks to "GitHub Pages" service.

The Theme

Starting from a blank page is neither easy nor particularly motivating for a personal project, so I chose to start from a theme. Indeed, the Hugo site offers many themes. That’s why I chose the one developed by Eddie Webbinaro: hugo resume.

Hosting

This site is hosted using the free GitHub Pages service.
