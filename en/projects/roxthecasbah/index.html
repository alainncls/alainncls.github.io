RoxTheCasbah

March 2019 - Present

Project's website

The context

Creation of a blog for my partner, RoxTheCasbah, to share her passion for knitting and sewing.

The technical solution

The first version

The first version of the site used Ghost, a complete blogging solution competing with WordPress. Based on an open-source and free tool, the blog was hosted on a small AWS machine.

The second version

In January 2021, I chose to switch to a static site generator, Hugo, in order to be able to host the generated pages for free.

The blog is thus deployed and hosted thanks to the free service GitHub Pages.
