Exporter facilement vos transactions

26 février 2025

Suivre son activité onchain peut être fastidieux, surtout si l’on veut un export rapide pour l’analyser. J’ai donc créé Transaction Exporter, un petit script Node.js qui récupère vos transactions et les exporte en CSV.

Ce qu’il fait

  • Récupère les transactions normales et internes

  • Génère un CSV clair et exploitable

  • Compatible avec Etherscan, Polygonscan et autres explorateurs similaires

Comment l’utiliser

  1. Clonez le repo : transaction-exporter

  2. Installez les dépendances : npm install

  3. Ajoutez votre clé d’API et l’URL de l’explorateur dans un fichier .env

  4. Lancez : npm run fetch <address> <handle>

Un outil simple, sans prise de tête, pour accéder rapidement à vos transactions.
